# 밀턴 에릭슨 전략적 커뮤니케이션 마스터 클래스 (Advanced Detailed Curriculum)

본 커리큘럼은 밀턴 에릭슨의 실제 발화 데이터 7,411건에 대한 통계적, 계산언어학적 분석(HMM, Markov Chain, NLP)을 기반으로 설계된 8회차(회당 3시간, 총 24시간) 전문가 마스터 과정입니다.
각 세션은 **데이터 인사이트(수치적 근거) -> 심리적 메커니즘(이론) -> 에릭슨의 실제 스크립트 분석(DNA) -> 마이크로 템플릿 실습(체화)**의 4단계 구조로 촘촘하게 설계되었습니다.

## 📅 [1회차] 데이터로 증명된 에릭슨의 비밀: 라포(Rapport)의 과학
**"모든 변화의 80%는 완벽한 주파수 동기화에서 시작된다."**

### 📊 코어 데이터 (Core Data)
*   **HMM 대화 상태 분석**: 전체 대화의 **78.6%**가 '라포 형성기'에 할애됨.
*   **상태 회귀성 (Transition Probability)**: 저항 우회(71%), 은유 삽입(73%), 행동 촉구(74%) 등 어떤 고난도 기법을 사용한 후에도 에릭슨은 다시 라포(RAPPORT) 상태로 70% 이상의 확률로 돌아감.

### 🕒 세부 모듈 및 타임라인 (Total: 180min)
*   **[00:00 - 00:30] 오리엔테이션 및 데이터 시각화 브리핑**
    *   직관에 의존하던 대화를 알고리즘(AI)으로 분해하다.
    *   마스터 전략 네트워크(Master Strategic Network) 맵 읽는 법.
*   **[00:30 - 01:10] Pacing & Truisms (보조 맞추기와 보편적 사실)**
    *   **메커니즘**: 부정할 수 없는 보편적 사실(Truism)을 나열하여 뇌의 비판적 필터(Critical Factor) 우회하기.
    *   **패턴 분석 (`ERICKSON_TRUISM_CALIBRATION`)**: 
        *   *에릭슨 스크립트*: "어떤 사람이 당신을 똑바로 쳐다보고 있을 수도 있지만, 당신은 그들이 당신에게 주의를 기울이지 않고 있다는 것을 그들 자신도 모르는 사이에 알 수도 있습니다."
        *   *Formula Logic*: "그들(They) -> 주의를 기울이다(Pay attention)" 구조 분석.
*   **[01:10 - 01:40] 감각적 경험의 언어화 (Mirroring in Language)**
    *   **패턴 분석 (`ERICKSON_PACING_AND_VALIDATION`)**:
        *   내담자의 호흡, 미세한 근육 움직임 등 현재 감각을 실시간으로 읽어주기.
        *   *에릭슨 스크립트*: "당신의 왼손에 대해 조금 놀라셨나요?"
*   **[01:40 - 01:50] ☕ Break Time**

    *   **[실습 A] 3 Pacing + 1 Leading 작성법**: 현재 검증 가능한 사실 3가지를 말한 뒤, 상대가 받아들이길 원하는 제안 1가지를 자연스럽게 연결하는 문장 구조(Formula) 작성.
    *   **[실습 B] 2인 1조 상호 페이싱**: 상대방의 미세한 무의식적 신호(고개 끄덕임, 눈 깜빡임, 한숨)를 문장 속에 중계하듯 녹여내어 '완전한 이해'의 환상 구축하기.
